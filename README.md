## 🚀 Getting Started

### 📦 Dependencies

- next 13.4+
- node v18+
- npm v9.5+
- tailwind v3.3+

### 👉 Install Dependencies

```bash
npm install
```

### 👉 Development Command

```bash
npm run dev
```

### 👉 Build Command

```bash
npm run build
```
